# Welcome to NotaBox Documentation

NotaBox is a powerful note management system designed to help teams organize and access their knowledge base efficiently.

<Callout type="info">
  This documentation will guide you through all features and best practices for using NotaBox effectively.
</Callout>

## Getting Started

1. **Authentication**: Sign in using your credentials or continue as a guest
2. **Search**: Use the powerful search functionality to find your notes
3. **Filters**: Apply filters to narrow down your search results
4. **Records**: Create and manage your records with custom fields and tags

## Key Features

- **Smart Search**: Advanced search capabilities with suggestions and history
- **Custom Fields**: Add structured data to your records
- **Tagging System**: Organize records with customizable tags
- **File Attachments**: Support for various file types
- **Team Collaboration**: Share and manage records with your team
